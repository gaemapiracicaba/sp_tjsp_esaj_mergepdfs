# TJSP e-SAJ

<br>

O Sistema e-SAJ, do TJSP, possibilita exportar o processo de duas formas -- arquivo único ou peças separadas.
Quando exportamos por arquivo único, perde-se a navegação pelos *bookmarks* (índice/sumário) que é possível no sistema do e-SAJ.

Porém, quando se exporta por peças individualizadas, é possível concatena-las gerando esse índice.
